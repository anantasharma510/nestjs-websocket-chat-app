# Real-time Group Chat App (NestJS + Socket.IO)

A **real-time group chat application** built using **NestJS** and **Socket.IO**.  
This project demonstrates a multi-user chat system with **dynamic online count**, **join/leave notifications**, and **messages showing sender names**.

---

## Features

- Users must enter a **name** before joining the chat
- **Real-time messaging** across multiple users/tabs
- **Dynamic online user count** displayed in the UI
- **System notifications** for users joining or leaving
- Clean and responsive **HTML/CSS frontend**

---

## Tech Stack

- **Backend:** NestJS, Socket.IO
- **Frontend:** HTML, CSS, JavaScript
- **Realtime:** WebSockets via Socket.IO

1. **Clone the repository**

```bash

Install backend dependencies
npm install
Run the NestJS server
npm run start:dev
Open the frontend

Simply open the index.html file in your browser.

Make sure the server is running at http://localhost:4000.

How it works
The backend handles user connections, disconnections, and message broadcasts.
Each message includes the sender’s name.
The online count updates dynamically whenever users join or leave.
System messages notify everyone when a user enters or exits the chat.
